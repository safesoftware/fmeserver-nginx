FME Server is not responding.

Possible Reasons:

  • Tomcat (the FME Server web server) crashed because of low memory.
  • Tomcat is timing out because of high load.
  • A request to FME Server took too long to process.

Potential Resolutions:

  • The server may recover automatically within the next 10 minutes.
  • If the problem persists, please reboot your server through the FME Cloud UI.
